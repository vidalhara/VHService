VHTimer

open class VHTimer

You can use VHTimer to handle timers without memory leak with delegate or completion block.

let timer = VHTimer({
    // Do action
})

timer.start(sec: 5.0, isRepating: true)

  • start(sec:isRepeating:)

    Creates a timer and schedules it on the current run loop in the default mode. After seconds have elapsed, the timer fires, sending the message with delegate or completion block.

    Declaration

    Swift

    open func start(sec: Double = 0.3, isRepeating: Bool = true)

    Parameters

    sec

    The number of seconds between firings of the timer. If sec is less than or equal to 0.0 this timer will stop.
    isRepeating

    If true, the timer will repeatedly reschedule itself until stopped. If false, the timer will be invalidated after it fires.
